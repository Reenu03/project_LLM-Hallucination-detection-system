Large Language Models can sometimes produce responses that sound correct but are actually inaccurate or fabricated. 
This project detects such cases using a combination of NLP techniques, model-based verification, and rule-based checks.

Key Features:
 Detects hallucinations in LLM-generated text
 Confidence scoring for each response
 Supports multiple detection strategies (rule-based + model-based)
 Fast and scalable pipeline
 Evaluation metrics for performance analysis
 Modular design for easy extension
